Raven's Developer Portfolio
Overview

This is a personal developer portfolio built using Next.js, React, Tailwind CSS, React Three Fiber, Shadcn UI, and an integrated AI SDK. The portfolio serves as both a showcase of projects, skills, and experience, and as an interactive platform with an AI-powered assistant for visitors.

The project follows best practices for a Next.js src/ structure, ensuring clear organization and maintainability.

Core Features
Hero Section: Introduction with dynamic visuals.
About Me: Short bio, skills, and professional background.
Projects Showcase: Display of portfolio projects with descriptions and links.
Skills: Highlights technical competencies.
3D Interactive Section: Built with React Three Fiber for immersive visuals.
AI Assistant / Chatbot: Interactive AI assistant powered by the AI SDK to guide visitors, answer questions, or provide personalized insights.
Contact Form: Enables visitors to reach out directly.
Blog (Optional): Space to share posts or updates.
Project Structure (Textual Overview)
src/
components/ # Reusable UI components
sections/ # Portfolio sections (Hero, About, Projects, etc.)
pages/ # Next.js pages
assets/ # Images, 3D models, icons, etc.
ai/ # AI assistant logic and SDK integration
styles/ # Global and component-specific styles
Notes for AI
The project uses Anthropic Skills, allowing Copilot to infer frameworks, libraries, and project structure automatically.
Key interactive functionality (like AI assistant and 3D visuals) is modularized for clarity.
Components are structured to separate UI, features, and AI logic for easier AI comprehension.
